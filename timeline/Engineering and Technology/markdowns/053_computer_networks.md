```mermaid
timeline
    title The History of Computer Networks
    section The 1960s: The Dawn of Networking
        1965 : First wide-area network (WAN) experiment
        1969 : ARPANET, the precursor to the Internet, goes online
    section The 1970s: The Rise of Protocols
        1972 : Email is invented
        1973 : Ethernet is developed by Robert Metcalfe
        1974 : TCP (Transmission Control Protocol) is developed
    section The 1980s: The Internet Takes Shape
        1983 : TCP/IP becomes the standard protocol for ARPANET
        1983 : Domain Name System (DNS) is introduced
        1989 : The World Wide Web is invented by Tim Berners-Lee
    section The 1990s: The Web Goes Public
        1991 : The first web page goes online
        1993 : Mosaic, the first graphical web browser, is released
        1995 : Commercialization of the Internet
    section The 2000s: The Wireless Era
        1999 : Wi-Fi (802.11b) standard is released
        2000s : Rise of broadband and Voice over IP (VoIP)
    section The 2010s: The Mobile and Cloud Era
        2010s : Proliferation of mobile devices and 4G networks
        2010s : Rise of cloud computing and software-defined networking (SDN)
```
