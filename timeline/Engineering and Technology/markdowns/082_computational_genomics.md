        1988 : The Human Genome Project is launched
    section The 1990s: The Internet and the Web
        1990 : The BLAST algorithm is published
        1995 : The first complete bacterial genome is sequenced
    section The 2000s: The Post-Genomic Era
        2001 : The first draft of the human genome is published
        2003 : The Human Genome Project is completed
        2005 : Next-generation sequencing (NGS) is introduced
    section The 2010s: Big Data and Cloud Computing
        2010s : The 1000 Genomes Project
        2010s : Rise of cloud computing for genomic data analysis
